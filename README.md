# Chatbot_NodeJs
Chatbot no Telegram com Node JS projeto curso DIO com Prof. Carlos Victor Gomes
Passo 1 criar uma conta no telegram, criar um bot pelo botfather do telegram, e copiar o token gerado para salvar no arquivo local index.js, instalar a biblioteca node-telegram-bot-api
passo 2 gerar um acesso de api no dialogflow do google, criar uma nova chave e salvar numa pasta json dentro do projeto, instalar a biblioteca dialogflow
passo 3 Gerar chave de acesso ao youtube pelo mesmo projeto gerado no dialogflow do google, copiar a chave de acesso e salvar num arquivo json, instalar a biblioteca youtube-node
passo 4 no dialog flow do google, criar uma intent com entrada de pedido e resposta para videos e salvar, apos isso nos arquivos especificos criar modulos de exportação para as funções a serem usadas em arquivos diferentes, dialogflow.js, index.js, youtube.js
